# 치지직 채팅 커스텀

치지직 채팅창의 텍스트 표시와 꾸미기 옵션을 내 브라우저에서만 토글하는 확장프로그램입니다.

## 채팅창 표시 옵션

치지직 채팅창에서 필요한 항목만 켜고 끌 수 있습니다.

- 닉네임
- 배지
- 타임스탬프
- 후원 랭킹

## 채팅 외형 옵션

채팅별 외형을 간단하게 변경할 수 있습니다.

- 채팅 박스 색상
- 큰 글씨
- 굵게

## 채팅 헤더 버튼

채팅 헤더의 작은 버튼으로 설정을 열거나 비로그인 채팅창으로 전환할 수 있습니다.

- 비로그인 채팅창은 보는 용도로만 사용할 수 있으며, 채팅 전송은 지원하지 않습니다.
- 현재 페이지에서 감지한 테마에 맞춰 화면을 표시합니다.
- 클린봇은 비활성 상태로 시작합니다.
- 설정 버튼과 비로그인 전환 버튼은 필요에 따라 켜고 끌 수 있습니다.

## 안내

이 확장프로그램은 네이버 또는 치지직의 공식 앱이 아니며, 네이버/치지직과 제휴된 도구가 아닙니다.

본 확장프로그램은 치지직과 관련이 없으며, 네이버, 치지직, CHZZK은 NAVER㈜의 등록상표입니다. 본 확장프로그램을 사용하여 발생하는 결과에 대한 모든 책임은 사용자에게 있습니다.

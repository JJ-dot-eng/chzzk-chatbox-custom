# CHZZK Chat UI Toggle

치지직 채팅창에서 내가 보는 화면의 표시만 조정하는 Chrome/Edge 확장 프로그램입니다.

## 기능

- 닉네임 표시 on/off
- 닉네임 앞 배지 표시 on/off
- `hh:mm` 타임스탬프 표시 on/off
- 채팅별 라운드 박스 표시 on/off
- 큰 글씨 on/off
- 굵게 on/off
- 채팅 박스 색상 선택 및 HEX 코드 입력
- 토글 상태를 브라우저 로컬 저장소에 저장

## 설치

### Chrome

1. `chrome://extensions`를 엽니다.
2. 개발자 모드를 켭니다.
3. 압축해제된 확장 프로그램을 로드합니다.
4. 이 프로젝트 폴더를 선택합니다.

### Edge

1. `edge://extensions`를 엽니다.
2. 개발자 모드를 켭니다.
3. 압축 풀린 항목 로드를 선택합니다.
4. 이 프로젝트 폴더를 선택합니다.

## 사용

치지직 페이지에서 확장 프로그램 아이콘을 누른 뒤 원하는 항목을 켜거나 끕니다.

이 확장 프로그램은 치지직 서버, 실제 채팅 데이터, 메시지 삭제/차단 기능을 변경하지 않습니다. 브라우저에 렌더링된 채팅 UI만 숨기거나 다시 보이게 합니다.

## 개발 메모

- Manifest V3 기반입니다.
- Chrome과 Edge에서 공통 지원되는 `chrome.*` 확장 API를 사용합니다.
- 치지직 DOM 구조가 바뀔 수 있으므로 content script는 selector 기반 탐지와 `MutationObserver` 재스캔을 함께 사용합니다.

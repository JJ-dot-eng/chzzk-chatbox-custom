# 치지직 채팅 커스텀

치지직 채팅창의 텍스트 표시와 꾸미기 옵션을 내 브라우저에서만 토글하는 확장프로그램입니다.

## 채팅창 표시 옵션

치지직 채팅창에서 필요한 항목만 켜고 끌 수 있습니다.

- 닉네임
- 배지
- 타임스탬프
- 채팅 외 패널

채팅 외 패널 표시를 끄면 후원 랭킹, 승부예측, 스포츠 중계 정보처럼 채팅 메시지가 아닌 패널을 숨깁니다.

## 채팅 외형 옵션

채팅창 외형을 간단하게 변경할 수 있습니다.

- 채팅 박스 색상
- 글씨 크기/닉네임 크기 조정(이모티콘 포함)
- 굵게

## 비로그인 채팅창

- 보는 용도로만 사용할 수 있으며, 채팅 전송은 지원하지 않습니다.
- 강퇴 당한 방송에서 채팅창을 구경 할 수있습니다.
- 전체화면일때, 화면에 표시되지 않습니다.

## 채팅 헤더 버튼

채팅 헤더의 작은 버튼으로 설정을 열거나 비로그인 채팅창, 미니 플로팅 채팅창으로 전환할 수 있습니다.

- 현재 페이지에서 감지한 테마에 맞춰 화면을 표시합니다.
- 클린봇은 비활성 상태로 시작합니다.
- 설정 버튼, 비로그인 전환 버튼, 미니 채팅 버튼은 필요에 따라 켜고 끌 수 있습니다.

## 미니 플로팅 채팅창 표시

미니 플로팅 채팅창은 현재 방송의 치지직 채팅을 페이지 안에 작게 띄웁니다.

- 미니 플로팅 채팅창은 드래그와 크기 조절을 지원하며, 채팅 입력과 전송은 치지직 원래 채팅창 안에서 처리됩니다.
- 채팅 보기 모드는 다른 사람 채팅 목록과 입력칸을 함께 표시합니다.
- 입력 전용 모드는 내가 보낼 채팅 입력/전송 박스만 표시합니다.
- 채팅 외 패널 표시를 끄면 미니 플로팅 채팅창에서도 승부예측, 스포츠 중계 정보 같은 비채팅 패널을 숨깁니다.
- 전체화면에서만 표시를 사용하면 미니 플로팅 채팅창이 전체화면에서만 표시됩니다.
- 최소화 버튼 클릭시 회색 말풍선 아이콘으로 전환됩니다.
- 비로그인 채팅 창 활성화시 표시되지 않습니다.

# 개인정보 처리방침

`치지직 채팅 UI 설정`은 치지직 채팅창의 표시 방식을 사용자의 브라우저 화면에서만 조정하는 Chrome/Edge 확장프로그램입니다.

이 확장프로그램은 네이버 또는 치지직의 공식 앱이 아니며, 네이버/치지직과 제휴되거나 승인된 도구가 아닙니다.

## 수집하는 정보

이 확장프로그램은 개인정보, 채팅 내용, 계정 정보, 방문 기록, 결제 정보, 위치 정보 등을 수집하지 않습니다.

## 전송하는 정보

이 확장프로그램은 수집한 데이터를 외부 서버로 전송하지 않습니다. 개발자 또는 제3자에게 사용자 데이터가 전송, 판매, 공유되지 않습니다.

## 저장하는 정보

닉네임, 배지, 타임스탬프, 채팅 박스, 글씨 크기, 굵기, 박스 색상 설정만 브라우저의 로컬 확장 저장소에 저장합니다. 채팅창이 처음 표시될 때 깜빡임을 줄이기 위해 같은 설정값을 치지직 페이지의 로컬 캐시에도 저장할 수 있습니다. 이 설정은 사용자의 브라우저 안에서만 사용됩니다.

## 동작 범위

이 확장프로그램은 치지직 채팅창의 DOM/CSS 표시만 변경합니다. 실제 방송, 서버 상태, 채팅 데이터, 메시지 삭제, 차단, moderation 동작에는 영향을 주지 않습니다.

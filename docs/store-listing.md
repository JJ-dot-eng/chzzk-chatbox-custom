# 스토어 등록 메모

## 기본 정보

- 이름: 치지직 채팅 UI 설정
- 짧은 설명: 치지직 채팅창 표시를 조정하고 우측 상단 아이콘으로 시크릿 채팅 팝업을 엽니다.
- 단일 목적: 치지직 채팅창 UI를 사용자가 보기 편하게 로컬에서 꾸미고 숨기는 기능을 제공합니다.

## 상세 설명

치지직 채팅창에서 닉네임, 배지, 타임스탬프 표시를 켜고 끌 수 있습니다. 채팅별 둥근 박스, 큰 글씨, 굵기, 박스 색상도 설정할 수 있고, 현재 방송 채팅창 우측 상단의 작은 아이콘으로 시크릿 모드 팝업을 바로 열 수 있습니다. 실험 옵션으로 기본 채팅창 위치에 비로그인 채팅 iframe을 표시할 수 있습니다.

모든 변경은 사용자의 브라우저 화면에서만 적용됩니다. 실제 방송, 채팅 서버, 채팅 메시지 삭제/차단/moderation 동작에는 영향을 주지 않습니다.

이 확장프로그램은 네이버 또는 치지직의 공식 앱이 아니며, 네이버/치지직과 제휴되거나 승인된 도구가 아닙니다.

## 권한 설명

- `storage`: 사용자가 선택한 표시 설정을 브라우저 로컬 저장소에 저장합니다.
- `activeTab`: 현재 열려 있는 치지직 탭에 설정 변경을 즉시 반영합니다.
- `scripting`: 확장 업데이트 후 오래된 content script가 남아 있을 때 현재 치지직 탭에 최신 스크립트를 다시 적용합니다.
- `https://chzzk.naver.com/*`, `https://*.chzzk.naver.com/*`: 치지직 페이지와 채팅 프레임에서만 UI 표시 설정을 적용합니다.

## 스크린샷

스토어 등록 화면에는 사용자가 제공한 스크린샷 3장을 업로드하면 됩니다. 현재 대화에 올린 채팅 화면, 텍스트 탭, 꾸미기 탭 스크린샷을 사용합니다.

- 채팅 화면 적용 예시
- 텍스트 탭 설정 화면
- 꾸미기 탭 설정 화면

스크린샷은 확장프로그램 zip 안에 넣는 파일이 아니라 스토어 등록 화면에서 별도로 업로드하는 자료입니다.
